Pollen

A powerful high-level programming language for microcontrollers that is modular, object-oriented, fun to use and easy to learn.

Designed for efficiency of people and hardware.

Pollen was designed to make microcontroller software development faster and more productive for people without compromising any efficiency of hardware. It's made for folks of all skill levels, devices of all vendors and projects of all kinds.

Clarity

Pollen's syntax is clear and clean by design. The language is simple to use and easy to learn.

Reusability

With core types and libraries for common functionality as well as support for modules, classes, and variable configurations, Pollen makes it easy to reuse code and write reusable software.

Portability

With Pollen, you get built in support for multiple hardware targets, as well as language constructs and command line options that make re-targeting your hardware app for new hardware easy.

Pollen and its core libraries are open-source software. It is in the cloud and you can run it right now.
